# AEGAD WordPress Project

## Project Information
- **Site**: https://ae-gad.com
- **Theme**: Dlist Child Theme
- **Last Updated**: November 21, 2025

## Contents
- `dlist-child-theme/` - WordPress child theme files
- `wp-config.php` - WordPress configuration

## Features
1. Geolocation Warning System
2. Language Switcher (6 languages: AR, EN, ES, FR, RU, TR)
3. Smart Map Detection
4. WPML Integration
5. AffiliateWP Integration

## Installation
1. Upload theme to `/wp-content/themes/`
2. Configure wp-config.php with your database details
3. Activate the theme in WordPress admin
4. Clear all caches
